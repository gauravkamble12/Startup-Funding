🚀 Startup Funding Dashboard

An interactive data analytics dashboard built with Streamlit, Pandas, and Plotly that provides comprehensive insights into startup funding activities, investor behavior, funding rounds, industry sectors, and city-wise investment trends.

📊 Features
Overall Analysis
Total funding raised across all startups
Highest-funded startup analysis
Average funding metrics
Total startup count
Monthly funding trends
Funding round distribution
Top funded cities
Top funded startups
Startup Analysis
Startup-specific funding details
Industry and location insights
Total funding received
Year-wise funding trends
Investor distribution analysis
Investor Analysis
Investor portfolio overview
Total investment amount
Recent investments
Top funded startups
Sector-wise investment distribution
Investment trend analysis
City distribution visualization

🚀 Installation
git clone https://github.com/gauravkamble12/startup-funding-dashboard.git

cd startup-funding-dashboard

pip install -r requirements.txt

streamlit run investment.py
